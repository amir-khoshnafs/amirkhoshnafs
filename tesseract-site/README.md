# Tesseract Site

A fast, dark-blue, glassy portfolio site with subtle RGB accents.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Key UX
- Responsive by default (mobile + desktop)
- Liquid-glass surfaces (backdrop blur + sheen)
- RGB hover ring accents
- Command palette: Ctrl/⌘ K


## Run on your LAN

```bash
npm run dev -- --hostname 0.0.0.0 --port 8000
```
Then open: `http://<LAN_IP>:8000`
